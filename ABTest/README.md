# A/B 实验

## [What Is A/B Test?](WhatIs.md)

## 场景
```text
有新的产品优化或者功能迭代，尤其是对用户使用路径或者便捷性影响较大的改动
精细化运营中，针对不同人群各种类型的活动尝试
推荐策略的优化或新增
算法版本迭代
版本灰度发布
```

## 流程

建立产品漏斗 
确定产品链路核心指标
观察指标，提出优化假设
设计A/B 实验方案
开发 A/B 实验
运行实验
实验数据分析
实验结论
