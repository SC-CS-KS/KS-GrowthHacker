# 北极星指标

“North Star Metric” 北极星指标，又叫做“OMTM" One metric that matters, 唯一重要的指标。  
产品现阶段最关键的指标，其实简单说来就是公司制定的发展目标，不同阶段会有不同的目标。  
之所以叫北极星指标，是因为这个指标一旦确立，就像北极星一样，高高闪耀在天空中，指引着全公司上上下下，向着同一个方向迈进。
